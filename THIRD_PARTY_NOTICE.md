# Third-Party Components

FaceFinder uses third-party projects and build dependencies, including Python, PySide6/Qt, InsightFace, ONNX Runtime, OpenCV, NumPy, SciPy, scikit-image, Pillow, NVIDIA CUDA-related runtime packages, Microsoft Visual C++ Redistributable, and Inno Setup.

Each component remains governed by its own license and redistribution terms. The FaceFinder license does not relicense any third-party component or pretrained model.

The repository intentionally excludes downloaded runtimes, DLLs, ONNX model files, generated installers, user photographs, embeddings, scan databases, and findings.
