# GitHub Upload Checklist

Before uploading this folder:

- Upload the extracted source files, not this ZIP as the only repository content.
- Keep `.gitignore`, `README.md`, `LICENSE`, `NOTICE`, and `THIRD_PARTY_NOTICE.md`.
- Do not add `runtime/`, `staging/`, `dist/`, `data/`, `Findings/`, `logs/`, downloaded models, DLLs, databases, or installers.
- Use GitHub Releases for the built installer.
- Review release files for personal paths and generate a SHA-256 checksum.
